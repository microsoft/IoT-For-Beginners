# השתמש בגלאי האובייקטים שלך בקצה

## הוראות

בפרויקט האחרון, פרסת את מסווג התמונות שלך לקצה. עשה את אותו הדבר עם גלאי האובייקטים שלך, ייצא אותו כמודל קומפקטי והפעל אותו בקצה, תוך גישה לגרסה שבקצה מהמכשיר ה-IoT שלך.
