# Поддержка

## Как сообщить о проблемах и получить помощь  

Этот проект использует GitHub Issues для отслеживания ошибок и запросов на новые функции. Пожалуйста, проверьте существующие проблемы перед созданием новой, чтобы избежать дублирования. Для новых проблем создайте запрос на исправление ошибки или добавление функции как новую задачу.

Если вам нужна помощь или у вас есть вопросы по использованию этого проекта, свяжитесь с нами, создав задачу в этом репозитории.

## Политика поддержки Microsoft  

Поддержка для этого **ПРОЕКТА или ПРОДУКТА** ограничивается перечисленными выше ресурсами.
