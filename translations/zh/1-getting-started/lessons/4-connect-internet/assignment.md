# 比较和对比 MQTT 与其他通信协议

## 指导

本课程介绍了 MQTT 作为一种通信协议。除此之外，还有其他协议，例如 AMQP 和 HTTP/HTTPS。

请研究这些协议，并将它们与 MQTT 进行比较和对比。考虑功耗、安全性以及在连接丢失时的消息持久性。
