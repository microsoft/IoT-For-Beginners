# 开始学习物联网

在本课程的这一部分中，您将了解物联网（IoT），学习基本概念，包括构建您的第一个“Hello World”物联网项目并连接到云端。这个项目是一个夜灯，当传感器测量到光线水平下降时，夜灯会亮起。
