# IoT 初學者課程 - 一套完整的教學計劃  

Microsoft 的 Azure Cloud Advocates 團隊很高興提供一套為期 12 週、共 24 節課的教學計劃，專注於 IoT 基礎知識。每節課包含課前與課後測驗、詳細的教學指引、解決方案、作業等內容。我們採用基於專案的教學法，讓您在實作中學習，這是一種被證明能有效掌握新技能的方法。  

這些專案涵蓋了從農場到餐桌的食物旅程，包括農業、物流、製造、零售和消費者等 IoT 裝置的熱門應用領域。

## 教學法  

在設計這套課程時，我們採用了兩個教學原則：確保課程是基於專案的，並且包含頻繁的測驗。完成這系列課程後，學生將能建構植物監測與灌溉系統、車輛追蹤器、智慧工廠系統（用於追蹤與檢查食品），以及語音控制的烹飪計時器，並學會 IoT 的基礎知識，包括如何撰寫裝置程式碼、連接雲端、分析遙測數據，以及在邊緣設備上運行 AI。  

透過專案導向的內容設計，學習過程變得更具吸引力，並能增強學生對概念的記憶。  

此外，課前的低壓力測驗能幫助學生專注於學習主題，而課後測驗則能進一步鞏固記憶。這套課程設計靈活有趣，可以完整學習，也可以選擇部分內容。專案從簡單開始，隨著 12 週的進展逐漸變得更具挑戰性。  

每個專案都基於學生和愛好者可獲得的真實硬體。每個專案都深入探討特定的專案領域，提供相關的背景知識。成為成功的開發者需要了解解決問題的領域，這些背景知識能幫助學生將 IoT 解決方案與學習內容應用於真實世界的問題情境中。學生不僅學會如何構建解決方案，還能理解其背後的原因，並對最終使用者有更深的體會。

我們提供了兩種 IoT 硬體選擇，您可以根據個人偏好、程式語言知識或學習目標來選擇。此外，我們還提供了「虛擬硬體」版本，適合那些無法獲得硬體或想在購買前多了解的學習者。您可以在[硬體頁面](./hardware.md)找到更多資訊和購物清單，包括我們合作夥伴 Seeed Studio 提供的完整套件購買連結。
