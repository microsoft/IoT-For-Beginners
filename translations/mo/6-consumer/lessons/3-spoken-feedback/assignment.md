# 取消計時器

## 說明

在上一課的作業中，你已經在 LUIS 中新增了一個取消計時器的意圖。在這次作業中，你需要在無伺服器的程式碼中處理這個意圖，向 IoT 裝置發送指令，然後取消計時器。

## 評分標準

| 評分項目 | 優秀 | 合格 | 需要改進 |
| -------- | ---- | ---- | -------- |
| 在無伺服器程式碼中處理意圖並發送指令 | 能夠成功處理意圖並向裝置發送指令 | 能夠成功處理意圖但無法向裝置發送指令 | 無法成功處理意圖 |
| 在裝置上取消計時器 | 能夠接收到指令並成功取消計時器 | 能夠接收到指令但無法取消計時器 | 無法接收到指令 |
