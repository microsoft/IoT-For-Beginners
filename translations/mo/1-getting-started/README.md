# 開始學習物聯網

在本課程的這一部分，你將會接觸到物聯網（IoT），並學習基本概念，包括建立你的第一個「Hello World」物聯網專案，將設備連接到雲端。這個專案是一個夜燈，當感測器測得的光線水平下降時，燈會亮起。
