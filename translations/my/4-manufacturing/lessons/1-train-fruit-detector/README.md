Machine learning рђЪрђг рђњрђ«рђЁрђћрђЁрђ║рђђрђГрђ» рђЋрђ╝рђ▒рђгрђёрђ║рђИрђюрђ▓рђЁрђ▒рђърђірђ║ - рђърђёрђ║рђЪрђг рђњрђ▒рђљрђгрђћрђ▓рђи рђърђГрђЉрђгрђИрђърђ▒рђг output рђЎрђ╗рђгрђИрђђрђГрђ» рђЁрђљрђёрђ║рђЋрђ▒рђИрђЋрђ╝рђ«рђИрЂі machine learning algorithm рђЪрђг рђњрђ▒рђљрђгрђЎрђЙ рђюрђ▒рђирђюрђгрђърђірђ║рЂІ рђЉрђГрђ»рђирђћрђ▒рђгрђђрђ║ рђърђёрђ║рђЪрђг рђњрђ«рђюрђ▒рђирђюрђгрђЉрђгрђИрђърђ▒рђг algorithm рђђрђГрђ» рђџрђ░рђЋрђ╝рђ«рђИрЂі *machine learning model* рђърђГрђ»рђирђЎрђЪрђ»рђљрђ║ *model* рђЪрђ»рђЂрђ▒рђФрђ║рђърђ▒рђг trained algorithm рђђрђГрђ» рђАрђърђ»рђХрђИрђЋрђ╝рђ»рђђрђг input data рђАрђърђЁрђ║рђЎрђ╗рђгрђИрђђрђГрђ» рђЉрђірђирђ║рђърђйрђёрђ║рђИрђЋрђ╝рђ«рђИ output рђАрђърђЁрђ║рђЎрђ╗рђгрђИрђђрђГрђ» рђЏрђЏрђЙрђГрђћрђГрђ»рђёрђ║рђърђірђ║рЂІ

> ­ЪјЊ Machine learning algorithm рђЪрђг рђњрђ▒рђљрђгрђЎрђЙ рђюрђ▒рђирђюрђгрђърђірђирђ║ рђюрђ»рђЋрђ║рђёрђћрђ║рђИрђЁрђЅрђ║рђђрђГрђ» *training* рђЪрђ»рђЂрђ▒рђФрђ║рђърђірђ║рЂІ input рђћрђ▓рђи рђърђГрђЉрђгрђИрђърђ▒рђг output рђЎрђ╗рђгрђИрђђрђГрђ» *training data* рђЪрђ»рђЂрђ▒рђФрђ║рђърђірђ║рЂІ

рђЦрђЋрђЎрђгрђАрђгрђИрђќрђ╝рђёрђирђ║рЂі рђърђёрђ║рђЪрђг unripe bananas рђЏрђ▓рђи рђЋрђ»рђХрђЎрђ╗рђгрђИрђђрђГрђ» training data рђАрђќрђ╝рђЁрђ║ input рђЋрђ▒рђИрђЋрђ╝рђ«рђИрЂі training output рђђрђГрђ» `unripe` рђЪрђ» рђърђљрђ║рђЎрђЙрђљрђ║рђђрђгрЂі ripe banana рђЋрђ»рђХрђЎрђ╗рђгрђИрђђрђГрђ» training data рђАрђќрђ╝рђЁрђ║ input рђЋрђ▒рђИрђЋрђ╝рђ«рђИ output рђђрђГрђ» `ripe` рђЪрђ» рђърђљрђ║рђЎрђЙрђљрђ║рђћрђГрђ»рђёрђ║рђърђірђ║рЂІ ML algorithm рђЪрђг рђњрђ«рђњрђ▒рђљрђгрђАрђЋрђ▒рђФрђ║ рђАрђЂрђ╝рђ▒рђЂрђХрђЋрђ╝рђ«рђИ model рђљрђЁрђ║рђЂрђ»рђђрђГрђ» рђќрђћрђ║рђљрђ«рђИрђЎрђірђ║рЂІ рђЉрђГрђ»рђирђћрђ▒рђгрђђрђ║ рђърђёрђ║рђЪрђг model рђђрђГрђ» рђАрђърђ»рђХрђИрђЋрђ╝рђ»рђђрђг banana рђЋрђ»рђХрђАрђърђЁрђ║рђљрђЁрђ║рђЋрђ»рђХрђђрђГрђ» input рђЋрђ▒рђИрђЋрђ╝рђ«рђИрЂі рђЋрђ»рђХрђЪрђг ripe рђюрђгрђИ unripe рђюрђгрђИрђєрђГрђ»рђљрђгрђђрђГрђ» рђЂрђћрђирђ║рђЎрђЙрђћрђ║рђИрђћрђГрђ»рђёрђ║рђЎрђірђ║рЂІ

> ­ЪјЊ ML рђЎрђ▒рђгрђ║рђњрђџрђ║рђЎрђ╗рђгрђИрЂЈ рђЏрђюрђњрђ║рђЎрђ╗рђгрђИрђђрђГрђ» *predictions* рђЪрђ»рђЂрђ▒рђФрђ║рђърђірђ║рЂІ

ML рђЎрђ▒рђгрђ║рђњрђџрђ║рђЎрђ╗рђгрђИрђЪрђг binary answer рђЎрђЋрђ▒рђИрђЋрђФрђўрђ░рђИрЂі рђАрђЁрђгрђИ probability рђЎрђ╗рђгрђИрђђрђГрђ» рђЋрђ▒рђИрђърђірђ║рЂІ рђЦрђЋрђЎрђгрђАрђгрђИрђќрђ╝рђёрђирђ║рЂі model рђљрђЁрђ║рђЂрђ»рђђрђГрђ» banana рђЋрђ»рђХрђљрђЁрђ║рђЋрђ»рђХрђђрђГрђ» input рђЋрђ▒рђИрђЋрђ╝рђ«рђИ `ripe` рђЪрђ» 99.7% рђћрђ▓рђи `unripe` рђЪрђ» 0.3% рђЂрђћрђирђ║рђЎрђЙрђћрђ║рђИрђћрђГрђ»рђёрђ║рђърђірђ║рЂІ рђърђёрђирђ║ code рђЪрђг рђАрђђрђ▒рђгрђёрђ║рђИрђєрђ»рђХрђИ prediction рђђрђГрђ» рђЏрђйрђ▒рђИрђЋрђ╝рђ«рђИ banana рђЪрђг ripe рђќрђ╝рђЁрђ║рђърђірђ║рђЪрђ» рђєрђ»рђХрђИрђќрђ╝рђљрђ║рђЎрђірђ║рЂІ

рђЋрђ»рђХрђЎрђ╗рђгрђИрђђрђГрђ» detect рђюрђ»рђЋрђ║рђЏрђћрђ║ рђАрђърђ»рђХрђИрђЋрђ╝рђ»рђърђ▒рђг ML рђЎрђ▒рђгрђ║рђњрђџрђ║рђђрђГрђ» *image classifier* рђЪрђ»рђЂрђ▒рђФрђ║рђърђірђ║ - label рђљрђЋрђ║рђЉрђгрђИрђърђ▒рђг рђЋрђ»рђХрђЎрђ╗рђгрђИрђђрђГрђ» рђЋрђ▒рђИрђЋрђ╝рђ«рђИрЂі рђњрђ« label рђЎрђ╗рђгрђИрђАрђЋрђ▒рђФрђ║ рђАрђЂрђ╝рђ▒рђЂрђХрђђрђг рђЋрђ»рђХрђАрђърђЁрђ║рђЎрђ╗рђгрђИрђђрђГрђ» рђЂрђйрђ▓рђЂрђ╝рђгрђИрђърђірђ║рЂІ

> ­ЪњЂ рђњрђ«рђЪрђгрђЪрђг рђАрђюрђйрђћрђ║рђЏрђГрђ»рђИрђЏрђЙрђёрђ║рђИрђЁрђйрђг рђќрђ▒рђгрђ║рђЋрђ╝рђЉрђгрђИрђЂрђ╝рђёрђ║рђИрђќрђ╝рђЁрђ║рђЋрђ╝рђ«рђИрЂі label рђЎрђљрђЋрђ║рђЉрђгрђИрђърђ▒рђг output рђЎрђ╗рђгрђИрђђрђГрђ» рђЎрђюрђГрђ»рђАрђЋрђ║рђърђ▒рђг unsupervised learning рђђрђ▓рђирђърђГрђ»рђирђърђ▒рђг рђЎрђ▒рђгрђ║рђњрђџрђ║рђЎрђ╗рђгрђИрђђрђГрђ»рђюрђірђ║рђИ рђюрђ▒рђирђђрђ╗рђёрђирђ║рђћрђГрђ»рђёрђ║рђърђірђ║рЂІ ML рђАрђђрђ╝рђ▒рђгрђёрђ║рђИрђЋрђГрђ»рђЎрђГрђ»рђюрђ▒рђирђюрђгрђюрђГрђ»рђЋрђФрђђ [ML for beginners, Machine Learning рђАрђђрђ╝рђ▒рђгрђёрђ║рђИ 24 рђърђёрђ║рђЂрђћрђ║рђИрђЁрђг](https://aka.ms/ML-beginners) рђђрђГрђ» рђђрђ╝рђірђирђ║рђЋрђФрЂІ
