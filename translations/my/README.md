# IoT for Beginners - သင်ခန်းစာများ

Microsoft မှ Azure Cloud Advocates များသည် IoT အခြေခံများအကြောင်း 12 ပတ်၊ 24 သင်ခန်းစာများပါဝင်သော သင်ခန်းစာများကို ပေးဆောင်ရန် ဝမ်းမြောက်ဝမ်းသာဖြစ်နေပါသည်။ သင်ခန်းစာတစ်ခုစီတွင် သင်ခန်းစာမတိုင်မီနှင့်ပြီးနောက် စမ်းမေးခွန်းများ၊ သင်ခန်းစာကိုပြီးစီးရန် ရေးသားထားသောညွှန်ကြားချက်များ၊ ဖြေရှင်းချက်၊ လုပ်ငန်းတာဝန်များနှင့် အခြားအရာများပါဝင်သည်။ ကျွန်ုပ်တို့၏ ပရောဂျက်အခြေခံ သင်ကြားမှုနည်းလမ်းသည် တည်ဆောက်မှုဖြင့် သင်ယူရန် ခွင့်ပြုသည်၊ အတတ်ပညာအသစ်များကို 'တင်းတိမ်' ဖြစ်စေသော အတည်ပြုထားသော နည်းလမ်းဖြစ်သည်။

ဒီပရောဂျက်များသည် စိုက်ပျိုးရေးမှ စတင်ပြီးစားပွဲထိ အစားအစာ၏ ခရီးကို ဖော်ပြထားသည်။ ဤတွင် စိုက်ပျိုးရေး၊ လိုဂျစ်စတစ်၊ ထုတ်လုပ်မှု၊ လက်လီနှင့်စားသုံးသူ - IoT စက်ပစ္စည်းများအတွက် လူကြိုက်များသော စက်မှုလုပ်ငန်းဧရိယာများပါဝင်သည်။

## သင်ကြားမှုနည်းလမ်း

ဒီသင်ခန်းစာများကိုတည်ဆောက်ရာတွင် ကျွန်ုပ်တို့သည် ပရောဂျက်အခြေခံဖြစ်စေရန်နှင့် မကြာခဏ စမ်းမေးခွန်းများပါဝင်စေရန် သင်ကြားမှုနည်းလမ်းနှစ်ခုကိုရွေးချယ်ထားသည်။ ဒီစီးရီးအဆုံးတွင် ကျောင်းသားများသည် အပင်စောင့်ကြည့်ခြင်းနှင့်ရေချိုးစနစ်၊ ယာဉ်စောင့်ကြည့်စနစ်၊ စက်ရုံစမတ်စနစ်၊ အသံထိန်းချိန်စနစ်ကိုတည်ဆောက်ပြီး IoT အခြေခံများကိုသင်ယူပြီး device code ရေးခြင်း၊ cloud နှင့်ချိတ်ဆက်ခြင်း၊ telemetry ကိုခွဲခြမ်းစိတ်ဖြာခြင်းနှင့် AI ကို edge တွင် run လုပ်ခြင်းကိုသင်ယူမည်ဖြစ်သည်။

ပရောဂျက်များနှင့်အညီအကြောင်းအရာကိုသေချာစေရန် ကျောင်းသားများအတွက် ပိုမိုစိတ်ဝင်စားစေပြီး အကြောင်းအရာများကိုမှတ်မိစေရန်အထောက်အကူဖြစ်စေသည်။

ထို့အပြင် သင်ခန်းစာမတိုင်မီ စမ်းမေးခွန်းသည် ကျောင်းသား၏အာရုံစိုက်မှုကို သင်ခန်းစာအကြောင်းသို့ ဦးတည်စေပြီး သင်ခန်းစာပြီးနောက် စမ်းမေးခွန်းသည် အကြောင်းအရာများကို ပိုမိုမှတ်မိစေရန်အထောက်အကူဖြစ်စေသည်။ ဒီသင်ခန်းစာများကို အပြည့်အစုံသို့မဟုတ် အစိတ်အပိုင်းအလိုက်ယူနိုင်သော Flexible ဖြစ်စေရန်နှင့် ပျော်ရွှင်စေရန် ဒီသင်ခန်းစာများကိုဒီဇိုင်းဆွဲထားသည်။ ပရောဂျက်များသည် သေးငယ်ပြီး 12 ပတ်အတွင်းပိုမိုရှုပ်ထွေးလာမည်ဖြစ်သည်။

ပရောဂျက်တစ်ခုစီသည် ကျောင်းသားများနှင့် hobbyist များအတွက်ရရှိနိုင်သော hardware ကိုအခြေခံထားသည်။ ပရောဂျက်တစ်ခုစီသည် ပရောဂျက် domain ကိုအနက်ရှင်းပြုလုပ်ပြီး သက်ဆိုင်ရာနောက်ခံအချက်အလက်များကိုပေးသည်။ IoT developer အဖြစ်ပြဿနာများကိုဖြေရှင်းရန် domain ကိုနားလည်ခြင်းကအထောက်အကူဖြစ်စေသည်။ ကျောင်းသားများသည် IoT ဖြေရှင်းချက်များနှင့်သင်ယူမှုများကို အမှန်တကယ်ပြဿနာများနှင့်ဆက်စပ်စဉ်းစားနိုင်ရန်အထောက်အကူဖြစ်စေသည်။ 

## Hardware

ပရောဂျက်များအတွက် IoT hardware ကို သင့်ရဲ့ကိုယ်ပိုင်စိတ်ကြိုက်၊ programming language အတတ်ပညာ၊ သင်ယူမှုရည်ရွယ်ချက်များနှင့်ရရှိနိုင်မှုအပေါ်မူတည်ပြီးရွေးချယ်နိုင်သည်။ Hardware မရရှိနိုင်သူများအတွက် 'virtual hardware' version ကိုလည်းပေးထားပြီး ဝယ်ယူမှုမလုပ်မီပိုမိုလေ့လာလိုသူများအတွက်လည်းအသုံးပြုနိုင်သည်။ [hardware page](./hardware.md) တွင် 'shopping list' ကိုဖတ်ရှုနိုင်ပြီး Seeed Studio မှအပြည့်အစုံ kit များကိုဝယ်ယူရန်လင့်ခ်များပါဝင်သည်။

ဤစာရွက်စာတမ်းကို [Docsify](https://docsify.js.org/#/) ကို အသုံးပြု၍ အွန်လိုင်းမရှိဘဲ အသုံးပြုနိုင်ပါသည်။ ဤ repo ကို fork လုပ်ပြီး [Docsify ကို install လုပ်ပါ](https://docsify.js.org/#/quickstart) သင့် local စက်တွင် ထည့်သွင်းပါ၊ ထို့နောက် repo ရဲ့ root folder တွင် `docsify serve` ကို ရိုက်ထည့်ပါ။ website ကို သင့် localhost ရဲ့ port 3000 တွင် ပြသပါမည်: `localhost:3000`။
