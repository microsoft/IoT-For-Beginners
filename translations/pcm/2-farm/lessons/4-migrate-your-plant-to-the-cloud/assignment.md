# Learn about cloud services

## Instructions

Clouds, like Azure wey Microsoft get, no be only compute dem dey rent give people. Di main kain cloud services wey dem dey offer include:

* Infrastructure as a service (IaaS)
* Platform as a service (PaaS)
* Serverless
* Software as a service (SaaS)

Make you sabi di different kain services dem, and explain wetin dem mean and how dem take different. Explain which of di services go make sense for IoT developers.

## Rubric

| Criteria | Exemplary | Adequate | Needs Improvement |
| -------- | --------- | -------- | ----------------- |
| Explain di different cloud services | Give clear explanation of all di 4 kain services | Fit explain 3 kain services | Fit only explain 1 or 2 of di services |
| Explain which service dey relevant for IoT | Talk which services dey relevant for IoT developers and why | Talk which services dey relevant for IoT developers but no talk why | No fit talk which services dey relevant for IoT developers |
