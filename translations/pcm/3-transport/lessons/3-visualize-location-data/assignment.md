# Deploy your app

## Instructions

Plenty ways dey wey you fit take deploy your app so you fit share am with di world, like to use GitHub pages or use one of di many service providers. One beta way to do am na to use Azure Static Web Apps. For dis assignment, build your web app and deploy am for di cloud by following [dis instructions](https://github.com/Azure/static-web-apps-cli) or watch [dis videos](https://www.youtube.com/watch?v=ADVGIXciYn8&list=PLlrxD0HtieHgMPeBaDQFx9yNuFxx6S1VG&index=3). 

One benefit wey dey for Azure Static Web Apps na say you fit hide any API keys for di portal, so use dis chance take refactor your subscriptionKey as variable and store am for di cloud.
