# Investigate function bindings

## Instructions

Function bindings dey allow your code save blobs for blob storage by returning dem from di `main` function. Di Azure Storage account, collection and other details dey configured for di `function.json` file.

When you dey work with Azure or other Microsoft technologies, di best place wey you fit find beta information na [di Microsoft documentation for docs.com](https://docs.microsoft.com/?WT.mc_id=academic-17441-jabenn). For dis assignment, you go need read di Azure Functions binding documentation to sabi how you go fit set up di output binding.

Some pages wey you fit check to start na:

* [Azure Functions triggers and bindings concepts](https://docs.microsoft.com/azure/azure-functions/functions-triggers-bindings?WT.mc_id=academic-17441-jabenn&tabs=python)
* [Azure Blob storage bindings for Azure Functions overview](https://docs.microsoft.com/azure/azure-functions/functions-bindings-storage-blob?WT.mc_id=academic-17441-jabenn)
* [Azure Blob storage output binding for Azure Functions](https://docs.microsoft.com/azure/azure-functions/functions-bindings-storage-blob-output?WT.mc_id=academic-17441-jabenn&tabs=python)

## Rubric

| Criteria | Exemplary | Adequate | Needs Improvement |
| -------- | --------- | -------- | ----------------- |
| Configure di blob storage output binding | Fit configure di output binding, return di blob and store am for blob storage successfully | Fit configure di output binding, or return di blob but no fit store am for blob storage successfully | No fit configure di output binding |
