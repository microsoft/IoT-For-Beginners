# Compare domains

## Instructions

Wen you dey create your object detector, you get choice to choose from different domains. Compare how dem dey work for your stock detector, and talk about which one dey give better result.

To change the domain, click the **Settings** button for the top menu, choose new domain, click the **Save changes** button, then retrain the model. Make sure say you test am with the new version of the model wey you train with the new domain.

## Rubric

| Criteria | Exemplary | Adequate | Needs Improvement |
| -------- | --------- | -------- | ----------------- |
| Train the model with a different domain | Fit change the domain and re-train the model | Fit change the domain and re-train the model | No fit change the domain or re-train the model |
| Test the model and compare the results | Fit test the model with different domains, compare results, and talk about which one better pass | Fit test the model with different domains, but no fit compare the results or talk about which one better pass | No fit test the model with different domains |
