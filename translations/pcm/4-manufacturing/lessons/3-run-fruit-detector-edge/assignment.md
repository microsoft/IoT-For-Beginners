# Run oda services for edge

## Instructions

No be only image classifiers fit run for edge, anytin wey dem fit package inside container fit deploy go IoT Edge device. Serverless code wey dey run as Azure Functions, like di triggers wey you don create for di earlier lessons, fit run inside containers, and so e go fit run for IoT Edge.

Choose one of di previous lessons and try run di Azure Functions app inside IoT Edge container. You fit find guide wey go show you how to do am using different Functions app project for dis [Tutorial: Deploy Azure Functions as IoT Edge modules on Microsoft docs](https://docs.microsoft.com/azure/iot-edge/tutorial-deploy-function?WT.mc_id=academic-17441-jabenn&view=iotedge-2020-11).

## Rubric

| Criteria | Exemplary | Adequate | Needs Improvement |
| -------- | --------- | -------- | ----------------- |
| Deploy Azure Functions app to IoT Edge | E fit deploy Azure Functions app to IoT Edge and use am with IoT device to run trigger from IoT data | E fit deploy Functions App to IoT Edge, but e no fit make di trigger fire | E no fit deploy Functions App to IoT Edge |
