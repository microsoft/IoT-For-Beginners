# Compare and contrast microcontrollers and single-board computers

## Instructions

Dis lesson talk about microcontrollers and single-board computers. Make table wey go compare and contrast dem, and write at least 2 reasons why you go use microcontroller instead of single-board computer, and at least 2 reasons why you go use single-board computer instead of microcontroller.

## Rubric

| Criteria | Exemplary | Adequate | Needs Improvement |
| -------- | --------- | -------- | ----------------- |
| Create table wey compare microcontrollers to single-board computers | Create list wey get plenty correct items wey compare and contrast | Create list wey get only small items | Fit only come up with one item, or no item to compare and contrast |
| Reasons for using one over the other | Fit give 2 or more reasons for microcontrollers, and 2 or more for single-board computers | Fit only give 1-2 reasons for microcontroller, and 1-2 reasons for single-board computer | No fit give 1 or more reasons for microcontroller or for single-board computer |
