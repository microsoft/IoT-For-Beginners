# 在邊緣設備上使用您的物件檢測器

## 指導說明

在上一個專案中，您已將影像分類器部署到邊緣設備。現在，對您的物件檢測器執行相同的操作，將其匯出為精簡模型並在邊緣設備上運行，從您的 IoT 設備訪問邊緣版本。

## 評分標準

| 評分項目 | 優秀 | 合格 | 需要改進 |
| -------- | ---- | ---- | -------- |
| 將物件檢測器部署到邊緣設備 | 能夠使用正確的精簡域，匯出物件檢測器並在邊緣設備上運行 | 能夠使用正確的精簡域並匯出物件檢測器，但無法在邊緣設備上運行 | 無法使用正確的精簡域，匯出物件檢測器，並在邊緣設備上運行 |
