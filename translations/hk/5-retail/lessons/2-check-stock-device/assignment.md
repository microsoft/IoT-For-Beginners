# 在邊緣設備上使用你的物件檢測器

## 指引

在上一個項目中，你已經將圖像分類器部署到邊緣設備。現在，對你的物件檢測器做同樣的操作，將其導出為一個精簡模型，並在邊緣設備上運行，從你的 IoT 設備訪問邊緣版本。

## 評分標準

| 評分標準 | 優秀 | 合格 | 需要改進 |
| -------- | ---- | ---- | -------- |
| 將物件檢測器部署到邊緣設備 | 能夠使用正確的精簡域，導出物件檢測器並在邊緣設備上運行 | 能夠使用正確的精簡域並導出物件檢測器，但無法在邊緣設備上運行 | 無法使用正確的精簡域，導出物件檢測器，並在邊緣設備上運行 |
