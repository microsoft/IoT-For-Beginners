# 開始學習物聯網

在這部分課程中，你將會接觸到物聯網（IoT），並學習基本概念，包括建立你的第一個「Hello World」物聯網項目，將設備連接到雲端。這個項目是一個夜燈，當感測器測量到光線水平下降時，燈會亮起。
