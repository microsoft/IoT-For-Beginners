# 支援

## 如何提交問題和獲取幫助  

此項目使用 GitHub Issues 來追蹤錯誤和功能請求。在提交新問題之前，請先搜尋現有問題以避免重複。對於新問題，請將您的錯誤或功能請求提交為一個新問題。

如需有關使用此項目的幫助和問題，請透過在此存儲庫中提出問題與我們聯絡。
