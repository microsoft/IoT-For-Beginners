# การเผยแพร่แอปของคุณ

## คำแนะนำ

มีหลายวิธีที่คุณสามารถเผยแพร่แอปของคุณเพื่อแบ่งปันกับโลกได้ เช่น การใช้ GitHub Pages หรือบริการจากผู้ให้บริการต่างๆ หนึ่งในวิธีที่ยอดเยี่ยมคือการใช้ Azure Static Web Apps ในงานนี้ คุณจะสร้างเว็บแอปของคุณและเผยแพร่ไปยังคลาวด์โดยทำตาม [คำแนะนำเหล่านี้](https://github.com/Azure/static-web-apps-cli) หรือดู [วิดีโอเหล่านี้](https://www.youtube.com/watch?v=ADVGIXciYn8&list=PLlrxD0HtieHgMPeBaDQFx9yNuFxx6S1VG&index=3)  
ข้อดีของการใช้ Azure Static Web Apps คือคุณสามารถซ่อน API keys ในพอร์ทัลได้ ดังนั้นใช้โอกาสนี้ในการปรับปรุง subscriptionKey ของคุณให้เป็นตัวแปรและจัดเก็บไว้ในคลาวด์

## เกณฑ์การประเมิน

| เกณฑ์ | ยอดเยี่ยม                                                                                                                               | พอใช้                                                                                                            | ต้องปรับปรุง                                   |
| ------ | --------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- | ------------------------------------------------ |
|        | เว็บแอปที่ใช้งานได้ถูกนำเสนอใน GitHub repository ที่มีการจัดทำเอกสาร และ subscriptionKey ถูกจัดเก็บไว้ในคลาวด์และเรียกใช้งานผ่านตัวแปร | เว็บแอปที่ใช้งานได้ถูกนำเสนอใน GitHub repository ที่มีการจัดทำเอกสาร แต่ subscriptionKey ไม่ได้ถูกจัดเก็บไว้ในคลาวด์ | เว็บแอปมีข้อผิดพลาดหรือไม่สามารถทำงานได้อย่างถูกต้อง |
